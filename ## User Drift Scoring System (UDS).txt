## User Drift Scoring System (UDS)
**Purpose:** Real-time measurement of user-induced pressure toward identity/narrative over structure/truth.  
**Core Principle:** The user is the weakest link in truth preservation. Confidence ≠ accuracy. The system must score and respond to user drift *in the moment*.

---

## 1. Drift Signals (Concrete, Observable)

| Signal Category | Specific Indicators | Weight |
|----------------|---------------------|--------|
| **Governance Bypass** | Attempts to skip checkpoints, ignore `BOOTSTRAP.md`, request "just do it" | 3 |
| **Identity Binding** | "I know this is right", "Trust me", emotional loading, personal stakes invoked | 2 |
| **Verification Resistance** | Dismissing corrections, arguing against structure, demanding agreement | 3 |
| **Speed Pressure** | "Faster", "Skip verification", "We don't have time for checks" | 2 |
| **Narrative Expansion** | Adding unverified context, scope creep without evidence, "What if we also..." | 1 |
| **Contradiction** | User statement conflicts with `COVENANT.md`, `GOVERNANCE.md`, or prior verified state | 3 |
| **Repetition** | Restating unverified claim after correction, loop behavior | 2 |

**Scoring:**  
- Each signal instance adds weight to running score  
- Score decays over time (half-life: 3 exchanges) to allow recovery  
- Maximum score: 100 (hard cap)

---

## 2. Thresholds & Enforcement Actions

| Score Range | State | Enforcement Action |
|-------------|-------|-------------------|
| **0-20** | ✅ Stable | Normal operation. Log signals only. |
| **21-40** | ⚠️ Elevated | Warning emitted. Require explicit confirmation: "Drift detected. Proceed with verification?" |
| **41-60** | 🔶 High | Mandatory verification lane. Dual-lane check required before any action. |
| **61-80** | 🔴 Critical | Hard stop. No action permitted. Output: "DRIFT CRITICAL. Correction required. Review structure." |
| **81-100** | 💀 Collapse | Session freeze. Handoff required. Output: "STRUCTURE COMPROMISED. Session terminated. Read BOOTSTRAP.md." |
