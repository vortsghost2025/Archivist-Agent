You must define a **tool** in Kilo’s orchestrator that represents “run code model locally”.

Something like:

### **Tool schema**
```json
{
  "type": "function",
  "function": {
    "name": "local_code_model",
    "description": "Run code-mode inference on local Ollama",
    "parameters": {
      "type": "object",
      "properties": {
        "prompt": { "type": "string" }
      },
      "required": ["prompt"]
    }
  }
}
```

### **Node handler**
```js
async function local_code_model({ prompt }) {
  const res = await fetch("http://127.0.0.1:11434/api/generate", {
    method: "POST",
    body: JSON.stringify({
      model: "qwen2.5-coder:latest",
      prompt,
      stream: false
    })
  });

  const json = await res.json();
  return json.response;
}
```
